# **App Name**: ProjectFlow

## Core Features:

- Project List: Displays a list of registered projects with client names. Clicking on a project opens its details.
- Project Details: Displays project information separated by environment and furniture, showing the status of each stage (Measurement, Cutting Plan, Material Purchase, Pre-Assembly) and the responsible team member.
- Project Registration: Modal to register new projects including client name, environment, and furniture details. Option to add multiple environments.
- Team Member Registration: Modal to register team members with their name and a color for tags/labels associated with them.
- Productivity Reports: Displays cards related to productivity, ongoing projects, completed projects, and pending tasks per team member.
- Progress Saver: Mechanism that allows a team member to easily persist any updates or progress reports. Clicking on save buttons ensures preservation of the current stage of work.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to inspire professionalism and productivity.
- Background color: Light grey (#F5F5F5) for a clean and modern look.
- Accent color: Soft teal (#4CAF50) for interactive elements and highlights.
- Font pairing: 'Poppins' (sans-serif) for headlines and 'PT Sans' (sans-serif) for body text. Poppins offers a contemporary precise feel; PT Sans has good readability.
- Left sidebar for main navigation (Projects, Register Projects, Register Team, Reports). Central area for content display based on selected navigation.
- Use clear and consistent icons for navigation and status indicators.
- Subtle transitions and animations when navigating between pages and updating status.
